HTML Introduction


HTML, short for HyperText Markup Language, serves as the fundamental markup language for webpage creation.

Employing tags, it structures content, encompassing elements such as headings, paragraphs, images, and links.

HTML documents adhere to a standard structure comprising <html>, <head>, and <body> sections.

Elements commonly possess attributes, and the <!DOCTYPE html> declaration signifies HTML5 usage.

The integration of semantic HTML elements contributes to enhanced content structure.

HTML forms the core of web development, collaborating with CSS for styling and JavaScript for interactivity.




HTML are versatile for organizing data on web pages, offering attributes like border for appearance control.

Techniques like colspan and rowspan enhance layout complexity.

CSS styling enables customization of borders, background colors, and alignment.

Enhanced accessibility is achieved with semantic HTML elements and the scope attribute.

Responsive design ensures tables adapt to various devices.

In summary, HTML tables provide an efficient solution for structuring and presenting data.

HTML Basic Example:


<!DOCTYPE html> <html> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

The HTML document itself begins with <html> and ends with </html>.

The visible part of the HTML document is between <body> and </body>

HTML Styles


Applied through HTML, shape the visual presentation of elements. Methods include inline, internal, and external styles. Key concepts involve selectors, properties, values, inheritance, specificity, responsive design, external libraries, and animations.


Example:


<!-- Inline Style --> <p style="color: blue; font-size: 16px;">This is a blue paragraph with a font size of 16 pixels.</p> <!-- Internal Style --> <head> <style> p { color: green; font-size: 18px; } </style> </head> <body> <p>This is a green paragraph with a font size of 18 pixels.</p> </body> <!-- External Style --> <!--HTML FILE--> <link rel="stylesheet" type="text/css" href="style.css"> <p>This is a red paragraph with a font size of 20 pixels.</p> <!--CSS FILE--> <p>p { color: red; font-size: 20px; } </p>

HTML Colors


Colors in HTML, vital for web design, can be specified via names, hexadecimal notation, RGB values, RGBA values, and HSL/HSLA values. Understanding these methods empowers developers to create visually appealing designs.


Example:


<!-- Color Names --> <p style="color: red;">This text is in red.</p> <!-- Hexadecimal Notation --> <p style="color: #00ff00;">This text is in green using hexadecimal notation.</p> <!-- RGB Values --> <p style="color: rgb(255, 0, 0);">This text is in red using RGB values.</p> <!-- RGBA Values --> <p style="color: rgba(0, 0, 255, 0.5);">This text is in semi-transparent blue using RGBA values.</p> <!-- HSL Values --> <p style="color: hsl(120, 100%, 50%);">This text is in a shade of green using HSL values.</p>

HTML Tables


HTML tables organize data in rows and columns. Elements like <table>, <tr>, <th>, and <td> structure tables. Additional attributes, column/row spanning, CSS styling, accessibility considerations, and responsive design are key aspects of effective table usage.


Example:
<table> <thead> <tr> <th>name</th> <th>age</th> </tr> </thead> <tbody> <tr> <td>James</td> <td>19</td> </tr> <tr> <td>April</td> <td>18</td> </tr> </tbody> </table>
